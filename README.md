# Smart Manufacturing ML-MAS

A Machine Learning driven Multi-Agent System for dynamic resource allocation in smart manufacturing supply chains.

## Overview

This project simulates a smart manufacturing ecosystem where factories, warehouses, suppliers, and logistics units operate as autonomous agents. The agents use machine learning and reinforcement learning to optimize resource allocation under uncertain demand.

## Features

- Demand forecasting using ML
- Multi-agent supply chain simulation
- Reinforcement learning based resource allocation
- Performance evaluation and visualization

## System Architecture

Demand Data → ML Forecasting → Supply Chain Simulation → RL Agents → Resource Allocation → Evaluation

## Tech Stack

Python  
Machine Learning (Scikit-learn)  
Reinforcement Learning (Q-learning)  
Multi-Agent Systems  
Data Visualization (Matplotlib)
